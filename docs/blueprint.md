# **App Name**: Bharat Bazaar

## Core Features:

- Demo Login: Implement a demo login page with pre-set credentials for easy exploration of the app.
- Product Catalog: Create a browsable and searchable product catalog with categories, descriptions, and images.
- Streamlined Checkout: Design a simplified and user-friendly checkout process for quick and easy purchases.
- AI Chatbot Support: Integrate an AI-powered chatbot to provide instant customer support, answer queries, and guide users.
- Admin Panel: Develop an admin panel for managing products, categories, and user accounts.
- Product Management Database: Implement a database to efficiently manage product information and inventory.
- Order Management System: Develop a system for managing orders, tracking shipments, and processing payments.

## Style Guidelines:

- Primary color: Deep maroon (#800000) to evoke richness and tradition.
- Background color: Desaturated maroon (#F2E1D1), a cream shade, for a soft and inviting feel.
- Accent color: Gold (#D4A773), an analogous color, to add a touch of elegance and Indian cultural context.
- Body and headline font: 'Alegreya', a serif font, offering elegance and readability for longer descriptions, harmonizing tradition and contemporary appeal.
- Use culturally relevant icons in gold to represent categories and actions.
- Maintain a clean and organized layout with clear visual hierarchy for easy navigation.
- Incorporate subtle animations and transitions to enhance user experience and provide visual feedback.
